# python-smtp-server

### Create .env file
```shell
SERVER_PORT=your_port_number
```

### Create credentials.json file
```shell
{
  "email": {
    "address": "your_address@gmail.com",
    "password": "your_address_password",
    "host": "smtp.gmail.com",
    "port": 465
  },
  "server": {
    "api_key": "your_api_key"
  }
}
```
